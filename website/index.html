Features

What makes OmriCode different

Built for developers who want full control. No telemetry, no censorship, no cloud dependency.

⬢
100% Local & Private
Everything runs on your machine. Ollama, OpenAI, Anthropic — your choice. No data leaves without your permission.
◆
Multi-Editor Architecture
Standalone app as AI hub. Thin plugins for VS Code, Godot, Blender — all connecting via local HTTP API.
◈
15 Built-in Tools
Read/write/edit files, run bash, web search/fetch, grep/glob, terminal access, ask user, explain code, and more.
◇
Dual Tool Calling
Native function calling for supported models + SEARCH/REPLACE regex fallback for any model. Works with everything.
▣
RAG & Memory
Vector store with cosine similarity search, function-aware code chunking, comment indexing across 6 languages.
◉
Zero Content Filters
No safety guidelines, no censorship, no guardrails. You are in full control of what the model generates.
